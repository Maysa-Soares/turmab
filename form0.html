O atributo type="text" especifica o valor que o input recebe.

Type="button" cria um botão para o evento.

Type="checkbox" é usado para dar mais de uma opção para o usuário.

Type="date" gera um calendario que facilita a inserção da data.

Type="Email" identifica se o campo possui "@" e ".".

Type="file abre uma janela para envio de arquivo."

Type="number" determina que o input receba um valor numerico.

Type="password" gera uma proteção qu eimpede que a senha seja vista.

type="radio" é usado para dar mais de uma opção para o usuario.

type="range" cria uma barra de progresso.

type="search" cria uma barra de pesquisa.

type="tel" cria um campo de telefone.

type="tel" cria um botão para enviar o formulario.
